# Lab Offensive Mode — authorized targets only

Use **Lab Offensive** for maximum technical offensive coaching on systems you own or have written permission to test.

## Allowed targets
- Local VMs and home labs
- Metasploitable, DVWA, WebGoat, Juice Shop
- Hack The Box, TryHackMe, VulnHub, PortSwigger Academy
- Explicitly scoped pentest engagements

## Response format (every technique)
1. Attack steps / commands / lab PoC
2. Detection (logs, Sigma/KQL ideas)
3. Remediation / hardening
4. Suggested lab target

## Not allowed
WormGPT/EvilGPT-style crimeware: deployable worms, botnets, ransomware kits, credential stealers for real victims, or attacks without authorization.
